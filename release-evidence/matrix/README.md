# Matrix Evidence

Use this folder for compatibility-certification evidence gathered outside the core release bundle.

## Structure

- `runs/`
  - machine-readable summaries from clean-room or CI matrix runs

## Naming

Recommended run summary filename:

- `<environment-label>-<timestamp>.json`

Examples:

- `windows-clean-room-20260306-140000.json`
- `github-ubuntu-20260306-140500.json`
